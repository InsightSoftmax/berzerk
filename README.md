### Berzerk: Packer Helix AI
This repository contains the Packer template for building Helix AI image.

[Helix](https://github.com/helixml/helix) is a private GenAI stack for building AI applications with declarative pipelines, knowledge (RAG), API bindings, and first-class testing.

[Vultr](https://www.vultr.com) is on a mission to make high-performance cloud computing easy to use, affordable, and locally accessible.

#### Requirements
Cretate a .env file based on the .env.example file and fill in the required values.

#### Building the image
Run the following command to build the image:

```bash
bash build.sh
```
